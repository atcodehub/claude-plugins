# Backend Development Plugin

Generates Laravel backend development code.

## What It Does

Claude automatically uses this skill for backend work. Creates production-ready code with:

- Laravel framework
- PHP language
- Database integration
- API development

## Usage

```
"Create a REST API for a music streaming app"
"Build a backend for an AI security startup"
"Design a settings panel with dark mode"
```

Claude will choose a clear backend direction and implement production code with meticulous attention to detail.
